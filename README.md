🛒 GreenLifeStore (C# + SQL Server Management Studio)

GreenLifeStore is a desktop-based inventory and sales management system developed using C# (.NET) with SQL Server Management Studio (SSMS) as the database.

It is designed to help a retail store efficiently manage products, stock levels, customer details, and sales transactions. The system provides a user-friendly interface for daily operations, ensuring fast data entry, accurate record keeping, and reliable reporting.

The application improves business efficiency by automating manual processes and maintaining secure, structured data storage using SQL Server.
